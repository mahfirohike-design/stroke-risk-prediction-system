# Stroke Risk Prediction System

## Analisis dan Prediksi Risiko Penyakit Stroke Menggunakan K-Nearest Neighbor (KNN)

Aplikasi berbasis web untuk memprediksi risiko penyakit stroke berdasarkan data rekam medis pasien.

### Metodologi

Tahapan penelitian meliputi:

1. Data Preprocessing

   * Penghapusan atribut ID Pasien
   * Transformasi data kategorikal menjadi numerik
   * Pembersihan data

2. Pemisahan Fitur dan Label

3. Pembagian Data

   * Train-Test Split (80:20)

4. Pembangunan Model Machine Learning

   * MinMaxScaler
   * SMOTE
   * K-Nearest Neighbor (KNN)

5. Optimasi Hyperparameter

   * GridSearchCV

6. Evaluasi Model

   * Accuracy
   * Precision
   * Recall
   * F1-Score
   * Confusion Matrix

7. Deployment

   * Streamlit

### Teknologi

* Python
* Streamlit
* Scikit-Learn
* Imbalanced-Learn
* Pandas
* Matplotlib
